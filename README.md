Quick Start
===========

``` html
$ git clone git://github.com/sergeylukin/dotfiles.git ~/bin/dotfiles
$ cd ~/bin/dotfiles
$ source install.sh
$ git config --global user.name "Your Name"
$ git config --global user.email "your@email"
```
